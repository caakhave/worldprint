# Security Policy

Can You Geo is a small public geography game project. Please report suspected security issues privately instead of opening a public issue with exploit details.

## Reporting A Vulnerability

Email: `support@canyougeo.com`

Please include:

- A short description of the issue.
- The affected URL, feature, or repository file if known.
- Reproduction steps that avoid exposing secrets or private user data.
- Any screenshots with tokens, emails, or private data redacted.

Do not send passwords, full payment card numbers, API keys, database URLs, or other secret values.

## Scope

In scope:

- Can You Geo public web app behavior.
- Account, billing, challenge email, and game-save flows.
- Public repository configuration that could expose users or secrets.

Out of scope:

- Social engineering.
- Denial-of-service testing.
- Physical attacks.
- Automated vulnerability scanning that disrupts service.
- Issues requiring access to another user's account or private data without consent.

## Response

The owner/operator will review reports as availability allows, prioritize user-data and billing risks first, and avoid disclosing sensitive details publicly until the issue is resolved.
